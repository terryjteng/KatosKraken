Clean Up After the Fight

Lightweight prototype: pick up trash, mop puddles, and sweep glass after a chaotic bout. Minimal code, fast load, replayable.

Overview

After the match ends, someone has to clean the arena. This demo is a compact, browser-native prototype that highlights design and production chops without heavy assets.

Gameplay

Collect trash instantly, mop puddles with the Mop, and sweep shards with the Broom. Randomized layouts keep each run fresh.

Playable Demo (Simple Prototype)

Controls: WASD/Arrows to move · E to clean · Q to switch tool (Mop/Broom)

Score: 0
Time: 60s
Tool: Mop
Replayable: new layouts each run

This is a lightweight prototype for portfolio purposes (no trademarked fighter assets). Big Boss Cleanup can be imagined as either a fast-paced overhead 2D game or a more immersive 3D simulator. In the 2D version, you navigate a top-down arena after each brawl, using the right tools to clean up procedurally generated messes while dodging lingering hazards. In the 3D version, you step directly into the aftermath in first- or third-person, tackling debris, puddles, and battle damage with physics-driven tools for a more realistic, simulator-style experience.
